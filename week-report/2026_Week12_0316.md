### 💡 Weekly Tech Trends / 本周技术趋势

- **Feature Interaction Deepening / 特征交互持续深化**: CDNet and MGDIN from Alibaba both push beyond the "one-size-fits-all" feature interaction paradigm — CDNet addresses seq-context heterogeneity with dual-view selection, while MGDIN tackles feature sparsity heterogeneity with deferred interaction. Both achieve >2% CTR lifts with zero latency overhead / CDNet和MGDIN均突破统一特征交互范式，分别从序列-上下文异质性和特征稀疏异质性两个角度推进，线上CTR提升>2%且零额外延迟
- **Long-tail in LLM-based Rec / LLM推荐中的长尾问题**: EISAM provides the first systematic analysis of long-tail effects in LRS, distinguishing prior vs. data long-tail — a timely contribution as LLM-based recommendation gains traction / EISAM首次系统分析LRS中的长尾效应，区分预训练先验和数据两类长尾，为LLM推荐系统的工业落地提供重要理论支撑
- **Test-Time Enhancement for RAG / RAG推理时增强**: Adobe's Agentic RAG test-time strategies demonstrate that simple inference-time modules (contextualization + caching) can improve both accuracy and efficiency without retraining / Adobe的Agentic RAG展示了推理时简单模块（上下文提炼+缓存）即可提升精度和效率
- **Alibaba Feature Interaction Series / 阿里巴巴特征交互系列**: Two companion papers (CDNet + MGDIN) from the same group demonstrate systematic exploration of feature interaction challenges in large-scale e-commerce — likely targeting top venues / 同组两篇companion papers系统性探索电商特征交互挑战
- **Model-Based Debiasing from Meta / Meta模型化去偏框架**: MBD's paradigm shift from bucket-counting to learned distribution estimation (μ, σ²) enables real-time, unified debiasing across item/user/model bias dimensions — a significant practical advance for large-scale short video platforms / MBD从桶计数转向学习分布估计（μ,σ²），实现短视频平台物品/用户/模型三类偏差的实时统一去偏
- **Multimodal Enterprise Search at Apple / Apple多模态企业检索**: AMES demonstrates that multi-vector late interaction retrieval (previously research-only) can be deployed in production on standard Apache Solr, closing the infra gap / AMES证明多向量后期交互检索可在标准Solr生产部署，弥合研究与工程落地鸿沟
- **LLM+Finance with Retrieval from Sber / Sber金融交易检索增强**: FinTRACE's behavioral KB with white-box AutoWoE rules doubles zero-shot LLM performance (MCC 0.19→0.38) on financial transaction classification — a practical blueprint for LLM+domain-specific retrieval in regulated industries / FinTRACE通过白盒规则知识库将零样本LLM性能翻倍，为金融等受监管行业的LLM+检索提供实践蓝图
- **Location-aware Embedding for Ad Retrieval at Yahoo / Yahoo地理位置感知广告检索**: The worLd2vec framework's compositional embeddings (query+location vectors) solve the long-tail cold-start problem for 45% of rare local queries, achieving up to +20% precision@K / worLd2vec的组合嵌入（查询+位置向量）解决了45%稀有本地查询的冷启动问题
